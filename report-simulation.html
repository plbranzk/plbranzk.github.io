Advanced Simulation – Assignment Collection

Goal

This project presents a collection of consecutive assignments submitted for the university course “Advanced Simulation”. The objective was to iteratively develop, analyze, and refine a transport simulation model for Bangladesh. Each assignment progressively builds toward more complex, robust simulation logic and validation strategies.

Methods

The assignments applied multiple modeling techniques including:

  • Discrete-event simulation and Agent-based modeling for simulating system behavior over time.
  • Development of executable models in software tools (e.g., AnyLogic, NetLogo).
  • Use of experimental design to test model behavior and sensitivity to parameters.
  • Validation techniques including extreme condition testing and face validity checks.

Skills Strengthened

This series of simulations fostered practical competence in:

  • Model design under constraints and assumptions.
  • Iterative model refinement and abstraction for clarity and validity.
  • Translating theoretical system behavior into implementable models.
  • Communicating simulation results in concise written reports and diagrams.
  • Critical reflection on model boundaries, purpose, and uncertainty.
